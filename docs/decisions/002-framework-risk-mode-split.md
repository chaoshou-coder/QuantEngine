# 002: 入场框架与风控模式正交分离

日期: 2026-03-05
状态: accepted

## 背景

V4 策略将入场逻辑（5 框架 F1-F5）和风控逻辑（4 模式 baseline/conservative/standard/aggressive）设计为独立维度，可正交组合。

## 选项

- **选项 A**：正交分离（当前方案）
  - 优点：结构化实验设计，可独立测试入场和风控效果
  - 缺点：组合数量增多
- **选项 B**：耦合设计（每个策略包含固定风控）
  - 优点：简单
  - 缺点：无法独立评估入场 vs 风控的贡献

## 决定

保留选项 A。正交分离是正确的结构化设计。

## 后果

框架数 × 风控模式数的组合需要在筛选阶段控制。
baseline（无风控）模式在 PRD 要求下可能需要标记为"仅限内部测试"。
