# 004: Walk-Forward 窗口参数选择

日期: 2026-03-05
状态: research-needed

## 背景

当前 WF 参数：1 年训练 / 0.5 年验证 / 0.5 年步长 / 40% OOS。
这些参数由 AI 自主决定，未经行业最佳实践验证。

## 待研究

- 学术文献中 walk-forward 窗口的推荐比例
- 不同市场（外汇 M1/M5）下的最佳窗口尺寸
- 训练/验证/OOS 的比例是否有公认标准
- 是否应该支持多种窗口配置并对比

## 选项

- **选项 A**：保留当前默认值（1y IS / 0.5y OOS / 0.5y step）
  - 优点：已验证可运行
  - 缺点：学术研究推荐更长窗口
- **选项 B**：改用更长窗口（2y IS / 1y OOS / 1y step）
  - 优点：学术共识推荐，减少过拟合
  - 缺点：可用 WF 轮数减少
- **选项 C**：参数化，支持多套预设并对比
  - 优点：灵活，数据驱动
  - 缺点：增加实验复杂度

## 决定

采用选项 B+C 组合：默认改为 2y IS / 1y OOS / 1y step（学术推荐），但窗口参数必须可配置（不硬编码），便于按需调整。

研究来源：
- Springer 2025 推荐 5y IS / 1y OOS 用于组合配置策略
- 学术共识 IS/OOS 比例 60-70% / 15-20%
- 分钟级策略中，窗口参数本身应被视为超参数进行验证

## 后果

- 默认 WF 窗口变长，单次实验的 WF 轮数减少（取决于数据总长度）
- 需要在配置系统中暴露 train_years / test_years / step_years 参数
- 后续可引入多窗口对比报告
